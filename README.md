# CXCrashProtect

<font color=#00FFFF size=16>Crash 防护框架，防止常见如数组溢出、增删nil、找不到方法等导致APP crash。</font>


#### 使用说明

直接将文件夹`CXCrashProtect`拖入到项目中即可。

防护支持类型：

|  支持类型|  说明| 
| --- | --- |
|  NSArray|  | 
|  NSMutableArray|  |  
|  NSDictionary|  |  
|  NSMutableDictionary|  |  
|  NSString|  |  
|  NSMutableString|  |  
|  NSSelector|  支持实例方法、类方法防护|
